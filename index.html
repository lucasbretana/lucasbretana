Papers

ILUCTUS: A Library to Support Collaborative Data Processing

2017

Process big volumes of data requires sites with capacity to storage and process large amount of data. The distributed processing utilizes technologies that allow the share of resources and costs of processing, and requires a tool that make the communication between the application and processing power. The Tuple Space is a programming was created over those technologies and is retaken in this paper as an alternative to the development of application in a cloud base environments. The Library ILUCTUS is presented in this paper, as well as a case study of its basics operations and common applications.
